프로젝트 개요

  • 프로젝트 이름: 신입 개발자 구직 추천
  • 프로젝트 기간: 2024.03.25~2024.04.08
  • 기능적 요소:
    · 아이디 중복 체크기능 추가 비밀번호 암호화를 통해 보안 유지
    · 비밀번호 확인 후 회원정보 수정 가능
    · 키워드, 학력 , 지역, 정렬순으로 채용공고 검색
    · 사용자별로 관심있는 뉴스를 스크랩하고 이를 재확인 가능
    · 사용자가 스크랩한 공고를 토대로 이와 유사한 10개의 채용공고를 추천
    · 사용자는 게시판에서 게시글과 댓글 작성이 가능하고 작성자만 수정 삭제가 가능

프로젝트 진행 사항

  • · 프론트엔드 구현: 사람인 API 사이트의 템플릿을 HTML5, CSS3, JavaScript를 활용하여 UI구현
  • · 데이터베이스 설계: 사용자, 채용공고, 게시판, 댓글 데이터베이스 설계
  • · 데이터 수집: 사람인 API에서 채용공고 데이터를 가져옴
  • · 데이터 전달: DB에 저장된 공고 데이터를 쿼리문을 통해 화면에 전달하고 검색기능 구현
  • · 백엔드 및 MVC 구현: Spring Framework 5와 Apache Tomcat 9를 사용하여 서버 사이드 로직 및 MVC 아키텍처 구현
  • · Ajax를 활용해 채용공고 스크랩 기능 구현
  • · 쿠키를 활용해 로그인하고 스크랩을 했을 때 유사공고 추천 버튼이 나오도록 구현
  • · 연봉을 기준으로 유사도 비교 알고리즘을 활용해 스크랩한 공고와 유사한 공고의 PK값을 Flask로 전달
  • · Ajax로 내용을 요청받아 컨트롤러에서 쿼리문을 사용 VO에 유사공고 10개의 정보를 저장 화면에서 제공
